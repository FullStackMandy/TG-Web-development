Opdracht 1.

Er zijn drie manieren waarop je CSS in je HTML kunt integreren.

  1. Je kan CSS direct in je HTML schrijven, door je style attribute te gebruiken, binnen het html-element wat je wil gaan aanpassen.
  2. Je maakt een appart extern css style sheet en linkt er in je html naartoe. Die link plaats je in je meta informatie, dat is binnen je head tag.
  3. Je schrijft je css style sheet direct in je html intern.
