De syntax van css is opgebouwd uit: een sectector, property en value. De setector zegt het al, selecteer het html element wat je wil aanpassen. Dat kan een <h1>tag zijn of een <p>tag. Vervolgens geef je met een property aan wat je wilt gaan veranderen, zoals de kleur.
