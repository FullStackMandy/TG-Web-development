Pseudo elements

Assignment 1

Selector is written like this ::first-line. For example, if you want to change the color of the first line of your paragraph, you write it out like this p::first-line and than you can style it with any color you like. Selects the first line of each <p> element

You can have ::first-letter, or ::before, or ::after an element. For instance a smily before your header, or after etc.
