  1. An absolute file path is the full URL to a file.
    For example: <img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">. A relative file path points to a file relative to the current page, the file path points to a file in the images folder located at the root of the current web. For example: <img src="/images/picture.jpg" alt="Mountain">
